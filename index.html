Как запустить

Два простых сценария публикации

  1. GitHub Pages Залейте репозиторий на GitHub, включите Pages и выберите деплой через GitHub Actions.
  2. Свой домен В настройках Pages добавьте домен, а у регистратора направьте DNS на GitHub Pages.
  3. ZIP-архив Если хостинг просит архив, загрузите файл release/blacksquare-site.zip.
